Statement

A statement is a line of code commanding a task. JavaScript statements ends with ; But it is optional (it is recommended by many developers).

If we have to write more than one statement in one line, we have to use semicolon to separate each statement.

JavaScript ignores multiple whitespace between statements. Using one whitespace makes the code more readable. To break a long statement, best practice to break it after a operator.

There are four types of statements in JavaScript - Values, Operators, Expressions, Keywords, and Comments.

Block

To bind multiple JavaScript statements , we have to use { } curly brackets. Code inside those curly brackets are called Block or Block code.

Syntax

Syntax is a set of rules to construct a program. JavaScript Syntax is similar to C, C++.
